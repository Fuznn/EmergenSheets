This is a passion project, can't publish it yet since Apple Dev Program is very expensive.
You are free to replicate this app to install it on your Apple phone, distributing it is an obvious no.

Forgot your tablet or iPad at home that you use to practice? This solves that, you can crop parts of your PDF
files in order for them to be easily readable on a small screen. It is hands free and uses metronome timing.

I know this app concept exists, but I had planned machine vision for this in order for setup to be minimal, all
hopefully handled automatically by machine vision stuff.
